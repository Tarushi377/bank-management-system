# bank-management-system
🏦 Bank Management System
📌 Project Description

The Bank Management System is a software application designed to manage basic banking operations such as account creation, deposit, withdrawal, balance inquiry, and account management. This system helps automate manual banking tasks and provides secure handling of customer data.

🚀 Features
Simple and user-friendly interface
for GUI)
 Backend: MySQL
 Database Connectivity: JDBC
 Features: Account creation, transactions, balance inquiry, backend management
